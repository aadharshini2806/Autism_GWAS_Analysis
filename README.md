"Functional Genomics"

Identification of gene loci and the genes associated with the variants linked to Autism Spectrum Disorder (ASD)

GWAS Catalog accession: GCST010514 

Dataset: ASD_SPARK_EUR_iPSYCH_PGC 

Sample size: Varies between 55420 and 9070, possibly due to imputations, genotype unavailability and cohort-specific variations

Summary statistics for the ASD dataset was obtained from the GWAS catalog, and filtered for genome-wide significant variants(p<5e-8)
The raw data file is really huge, so it is not uploaded. 

URL: https://ftp.ebi.ac.uk/pub/databases/gwas/summary_statistics/GCST010001-GCST011000/GCST010514/ASD_SPARK_EUR_iPSYCH_PGC.tsv

The tcs file should be downloaded and loaded in R for analysis

Key Findings: There were no GO-enriched terms, possibly since only 25 genes were identified as statistically significant from the GWAS data.

The variants were mapped to genes across 4 significant loci, where the variants are closely associated with the genes and span only a very small portion of each locus, hence clustered in a single region.

Through a literature review, the gene functions have been identified. (Refer Autism_GWAS_Results document for detailed results)
